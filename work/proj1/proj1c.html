    Metadata Management

      Purpose

    • Needed to update our workflow for managing the metadata of uploaded BUTV content.
    • Converted from licensed software for managing metadata to in-house built software.

      Impact

    • The first part of the COM Video project was the upload app and this was the second part.
    • Now users have a streamlined workflow for uploading content and managing that content's metadata.
    • Metadata added or modified saves to a database, which gets read by BUTV Stream and updates accordingly.
